  • nGrinder Recorder는 윈도우즈에서만 제대로 내장된 브라우저를 사용한 레코딩을 지원합니다.
    • OS별 지원 기능 범위는 여기를 참고하세요.
    • 맥/리눅스 사용자는 별도의 브라우저에 프록시 셋팅을 하여 레코딩하여야 합니다.
      Recording Guide For Non Window User.을 참고하시기 바랍니다.
  • 모바일 기기의 트래픽을 레코딩 하려면 다음 절차를 따르세요.
    아이폰 안드로이드
    • 설정으로 이동
    • Wi-Fi로 이동
    • 프록시를 설정하고자 하는 Wi-Fi 선택
    • 하단의 HTTP 프록시를 "수동"으로 선택
    • 서버에 ${http_port.host}를 설정
    • 포트에 ${http_port.port?c}를 설정
    • 인증은 선택하지 않음
    • 어플리케이션 탭으로 이후 설정 메뉴를 탭
    • 무선 및 네트웍을 탭.
    • 무선 네트웍이 활성화 된것을 확인후, 프락시 서버를 설정하고자 하는 네트웍에 연결.
    • Wi-Fi 설정 메뉴를 탭.
    • 가능한 네트웍에서, 사용하고자 하는 네트웍을 탭
    • Wi-Fi 접속을 편집 - 아래로 스크롤하여, 프록시 사용을 선택
    • 프록시 호스트 이름을 ${http_port.host}로 설정
    • 프록시 포트를 ${http_port.port?c}로 설정
    • 저장을 탭
  • nGrinder Recorder는 다음을 레코딩 할 수 없습니다.
    • 브라우저에서 자바스크립트에 의해 생성된 쿠키
    • 브라우저에서 자바스크립트에 의해 동적으로 생성된 HTTP 호출 파라미터 생성 로직
    • 웹 소켓
  • nGrinder Recorder에 대한 사용방법은 레코더 설명서를 참조하시기 바랍니다.
  • nGrinder Recorder는 TCPProxy 와 JXBrowser 에 기반하고 있습니다.
