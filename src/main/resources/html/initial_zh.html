  • 有关在各个平台上支持的功能，请参考 录制器的限制。
  • 使用内置浏览器录制，只能在Windows上使用。
    • 如果要在Mac上使用录制器录制脚本，请使用mac系统中安装的其他浏览器进行录制
      详情请参考 Recording Guide For Non Window User.。
  • 如果要从手机等移动设备连接代理,请按如下配置:
    IOS Android
    • 打开"设置"
    • 进入"无线局域网"中，连接到代理所在的无线网络
    • 进入所连接的网络的详细设置，找到下面的HTTP代理设置,并选择手动
    • 设置 服务器为 ${http_port.host}
    • 设置 端口为 ${http_port.port?c}
    • 禁用鉴定
    • 打开所有应用,打开"设置"
    • 打开"无线和网络"
    • 确认无线网络已经打开,而且连接到代理所在的局域网
    • 连接以后，打开WLAN设置
    • 在可选择网络列表中, 找到连接的网络长按,直到出现菜单
    • 选择"修改网络"
    • 勾选"显示高级选项"，然后选择手动
    • 设置 服务器为 ${http_port.host}
    • 设置 端口为 ${http_port.port?c}
    • 保存退出
  • nGrinder录制器无法自动记录下面的内容:
    • 浏览器中由javascript生成的cookie
    • 浏览器中由javascript加密的密码
    • 由 Web Socket产生的通信
  • 详细内容请参考 脚本录制向导。
  • nGrinder录制器是基于 TCPProxy 和 JXBrowser开发。
